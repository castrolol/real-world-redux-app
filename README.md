# A Real World React + Redux App


To install dependencies
```sh
 $ yarn
```
or
```sh
 $ npm install 
```

To start app

```sh
  $ yarn start
```
or 
```sh
  $ npm start
```
 


To serve data, install json-server

```sh
 $ yarn global add json-server 
```
or
```sh
 $ npm -g install json-server 
```

To init the server

```sh
  $ yarn run server
```
or 
```sh
  $ npm run server
```
